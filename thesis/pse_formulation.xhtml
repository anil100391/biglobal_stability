Let

1.2 Parabolized Stability Equations

For different analysis type q ˜ and Θ take different form. For example, for a bi-global analysis q ˜ ( x,y,z )= q ˜ ( x,y ) and Θ = β z- ω t . For Orr-sommerfeld equations q ˜ ( x,y,z )= q ˜ (y) and Θ = α x+ β z- ω t .The introduction of a harmonic decomposition in x and z implies homogeneity of the base flow in these directions. These conditions, however, are too restrictive for complicated vortical flows and those around two-dimensional bodies. Consequently, the flow is assumed to be general in a two-dimensional plane, and mildly inhomogeneous in the third direction which is permitted by allowing the axial wavenumber to vary slowly. This leads to the parabolised stability equation concept, formally introduced by Herbert[key-1].
q'( x,y,z,t )= q ˜ ( x,y,z ) exp i Θ
where Θ = ∫ z 0 z β ( ξ )d ξ - ω t
∂ q' ∂ z ={ i β q ˜ + ∂ q ˜ ∂ z } exp i Θ
and
∂ 2 q' ∂ z 2 ={ ∂ 2 q ˜ ∂ z 2 +2i β ∂ q ˜ ∂ z - β 2 q ˜ +i d β dz q ˜ } exp i Θ
β is streamwise wavenumber complex in nature given as β ( ξ )= β r ( ξ )+i β i ( ξ ) .
It should be noted that q ˜ ( x,y,z ) depends very weakly on z coordinate hence, second derivatives w.r.t z will be neglected. Therefore,
∂ 2 q' ∂ z 2 ≈ { 2i β ∂ q ˜ ∂ z - β 2 q ˜ +i d β dz q ˜ } exp i Θ

2.0.1 Normalization condition

Implicit in this derivation is that the disturbance takes the form of a rapidly varying phase function and a slowly varying shape function. Under this assumption second derivatives w.r.t x can be neglected. For these properties to be satisfied, and to resolve the ambiguity of representing the disturbance component as the product of two functions of x , a normalization condition restricting rapid streamwise changes must be applied. In a manner analogous to that used by Bertolotti[key-2] in their analysis of the two-dimensional flat-plate boundary layer, a suitable normalization condition is
∫ y min y max ∫ z min z max u ˜ * ∂ u ˜ ∂ x dydz= ∫ y min y max ∫ z min z max 1 2 ∂ ∂ x | u ˜ | 2 dydz=0
* represents the complex conjugate.

2.1 Solution methodology for PSE

The solution to PSE can be done by treating x 0 as time and marching in x direction using Euler scheme as described below.
Equation to be solved is
( L 0 + L 1 ) q ˜ + L 2 ∂ q ˜ ∂ x +i d α dx L 3 q ˜ =0
with initial conditions,
q ˜ ( x 0 ,y,z )= q 0 ( y,z ), α ( x 0 )= α 0
and boundary conditions, (Not finding a way to write it better)
u ˜ ( x, y b , z b )= v ˜ ( x, y b , z b )= w ˜ ( x, y b , z b )=0 ∀ y b ∪ z b ∈ boundaries
It can be identified that LHS of normalization condition is a inner product of u ˜ and ∂ u ˜ ∂ x
1
Bold u represent vector velocity perturbation
. Defining b as
b= < u ˜ , ∂ u ˜ ∂ x > < u ˜ , u ˜ >
From the definition b represent component of u ˜ x along u ˜ . Therefore we can write ( ∂ u ˜ ∂ x is written concisely as u ˜ x )
u ˜ x =b u ˜ +r( x,y,z )
where r( x,y,z ) represent component of u ˜ x orthogonal to u ˜ .
Homogeneous part of above differential equation yield solution u ˜ ∼ exp ( bx ) , which suggests u ˜ may vary rapidly if b>0 ; contrary to assumption about u ˜ .
Above orthogonal decomposition helps us understand normalization condition as follows. Given the values of α j and u ˜ j , we solve equation (1.2.5) for u ˜ j+1 . Now b j+1 can be obtained from definition of b as
b j+1 = < u ˜ j , u ˜ j+1 - u ˜ j ▵ x > < u ˜ j , u ˜ j >
Now in case b j+1 is non-zero growth rate of u ˜ is of concern to us. Hence we update α j as α j (1) = α j (0) + b j+1 (0) . Bertolotti has shown in his paper[key-2] that such a procedure leads to reduction in updated value of b j+1 . We repeat this procedure unless b j+1 acquires a sufficiently small value within prescribed tolerances. Thus, normalization condition provides the unique value of α (x) that removes any exponential change in u ˜ measured by b. Then, u ˜ equals r( x,y,z ) that captures the slow, streamwise variation of u ˜ . Once convergence for b is achieved we march to next step.

Chapter 3 Extension to generic geometries

The core code “pse” will be written to work on a coordinate system ( x,y,z ) wherein a very standard geometry is present as shown in the diagram below. x will be the streamwise direction. The y and z boundaries are restricted to y= ± 1 and z= ± 1 . Any different geometry if required to be solved using “pse.c” first must be transformed to standard geometry using some transformation f given as
( x,y,z )=f( χ , η , ζ )
After transformation to standard geometry, the solution is done and result is transformed back to original geometry.
